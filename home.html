Front-End development

The front end of a website is the part that users can see and interact with. It includes the user interface and the graphic design. The front end is the client-side part of a web application.

Back-End development

The backend, or server side, of a website is the part that stores and organizes data. It also ensures that everything on the client side works. The backend communicates with the frontend, sending and receiving information to display as a web page.

FullStack development

Full stack development is the process of designing, creating, testing, and deploying a complete web application
